# AI Usage Journal


Tools used
- ChatGPT / GitHub Copilot for code scaffolding, prompts, and debugging

Prompts (examples)
- "Generate an Express.js API with CRUD endpoints for a tasks table using sqlite3."
- "Create a minimal frontend HTML/JS to interact with the tasks API."
- "Add user authentication (register/login) using bcrypt and JWT, and protect task endpoints."

Where AI helped
- Initial project structure and code snippets for `server.js`, `db.js`, and frontend `app.js`.
- Adding auth examples and token handling.
- Writing documentation templates and planning artifacts.

Manual work
- Wiring endpoints together, testing locally, adjusting SQL and JS, and adding UX tweaks.

Challenges
- Ensuring sqlite3 callbacks were correctly handled and JSON responses matched frontend expectations.

Major AI-generated outputs
- Scaffolding code for the Express server, SQLite setup, and frontend boilerplate.

Areas where manual intervention was required
- Adjusting SQL schema to add `user_id`, handling token storage in frontend, and refining error handling.
