# AI Task Manager — Vibe Coder Assignment

Minimal Task Management web app built for the CEDCOSS Vibe Coder assessment.

Setup

1. Install dependencies:

```bash
npm install
```

2. Run locally:

```bash
npm run dev
# or: npm start
```

3. Open http://localhost:3000

Project structure
- `backend/` — Express server and SQLite DB init
- `public/` — static frontend (HTML/JS/CSS)
- `docs/` — planning, AI journal, progress log, reflection

Docs: see `docs/` for planning and logs.
